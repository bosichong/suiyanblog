今天，我的第一个开源 OpenClaw 技能正式上线 ClawHub。

## 为什么做这个

写代码的人都有过这种时刻：

洗澡时突然想到一个绝妙的解决方案，想记下来，但手机不在手边。等找到手机，想法已经模糊了。

灵感这东西，保质期极短。

所以做了 **Flash Thoughts（闪念）**，核心功能就四个字：**说话即记**。

```
你：帮我记下来，我想给每种博客程序写一个智能体技能
Agent：记好了！写入了 2026-03-20.md
```

不用打开 App，不用切窗口，对着 Agent 说一句话就行。

## 技术实现

- 按日期存储，每天一个 Markdown 文件
- 每个闪念用分隔线隔开，互不干扰
- 纯文本格式，grep 可搜，数据永远属于你
- 核心脚本不到 150 行 Python，零外部依赖

## 安装方式

**人类安装**：

```bash
# ClawHub 一键安装
clawhub install flash-thoughts

# 或者 Git Clone
git clone https://github.com/bosichong/flash-thoughts.git
cp -r flash-thoughts ~/.openclaw/workspace/skills/
```

**智能体安装**：

直接告诉你的 Agent："安装 flash-thoughts 技能"
