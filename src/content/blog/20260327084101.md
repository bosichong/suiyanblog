iFlow CLI 要关了。意料之中，又有点可惜。

2026年3月20日停止维护，4月17日正式关闭。官方公告说得很委婉，但意思很明确：这个免费的好东西，用不成了。

作为国内最早一批能用的终端 AI 编程助手，iFlow CLI 给了不少惊喜。内置 GLM、Kimi、MiniMax 等国产大模型，支持 Skills、Hooks、MCP，开箱即干。对于偶尔写写脚本的人来说，免费额度够用，体验也不错。

但免费午餐总有结束的一天。这就是云服务的 `rm -rf /`，没有备份，没有回滚。

## 换个姿势继续薅

官方推荐迁移到 Qoder 平台，但我找了一圈，发现阿里自家的 **Qwen Code** 更对胃口。

地址：https://qwen.ai/qwencode

**为什么选它**：

1. **免费额度实在**。注册就送，日常写代码够用，不像某些产品送你 10 次调用就没了。
2. **模型能打**。内置 Qwen3-Coder，编程能力比肩 Claude，登顶全球开源模型阵营。
3. **CLI + IDE 双形态**。终端和 VS Code 都能用，不用切到浏览器写代码，体验连贯。

## 快速上手

### 1. 安装

Linux/macOS 一键安装：

```bash
bash -c "$(curl -fsSL https://qwen-code-assets.oss-cn-hangzhou.aliyuncs.com/installation/install-qwen.sh)"
```

Windows 用户去官网看文档，也简单。

### 2. 配置

安装后运行 `qwen`，按提示登录阿里云账号或配置 API Key。

### 3. 开始用

在终端输入 `qwen`，然后描述你的需求：

```
写一个 Python 脚本，遍历当前目录下所有 .md 文件，统计字数并排序输出
```

回车，代码出来。复制粘贴改改就能用。

## 几个使用技巧

**描述要具体**。越具体，生成质量越高。

```
# 差
写个爬虫

# 好
用 Python 的 requests 库写一个爬虫，目标网站是 https://example.com，提取所有 <h2> 标签的内容，保存到 result.json，要有异常处理和重试逻辑
```

**学会追问**。第一版代码不完美？继续让它改：

```
加上日志输出，每爬取 10 个页面打印一次进度
```

**别盲目复制**。AI 写的代码要自己过一遍，特别是涉及到网络请求、文件操作的部分。逻辑对不对，异常处理有没有，心里要有数。

## 写在最后

iFlow CLI 倒了，Qwen Code 接上。工具换来换去，核心还是那一套：**让 AI 干重复劳动，把精力留给真正有价值的问题**。

免费的东西，能用一天是一天。等哪天 Qwen Code 也开始收费，再找下一个就完事了。程序员的世界里，永远不缺 Plan B。
