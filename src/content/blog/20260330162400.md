十天，足够一些事情发生与改变……

经过半个月左右的密集使用，OpenClaw的新鲜感已过。主要表现为：

1. 由于自身的需求有限，OpenClaw的使用也达到了极限，并不是龙虾达到了极限，是人类的想象力达到了极限，不知道用它干嘛了。
2. 疲劳期，不停地安排任务，OpenClaw不停地犯错，不停地纠正记忆，很疲惫。
3. 期望过高造成的失落感随之而来，与OpenClaw的合作整体能力不及预期。

总之，目前陷入了一个新鲜感已过的阶段，估计很多人都会在密集使用后有这种奇怪的感觉吧。

## 工具箱 · Toolbox

一些好用的工具、站点、开源代码推荐，以及一些我个人的使用感受。

### Vibe Coding 指南

> 项目地址：https://github.com/2025Emma/vibe-coding-cn
> ⭐ 17,985 stars

**一个与 AI 结对编程的工作方法论指南**，教你如何正确地和 AI 一起写代码，把想法变成现实，而不是让 AI 把代码库搞成一团乱麻。

**核心理念**：规划就是一切。谨慎让 AI 自主规划，否则代码库会变成无法管理的乱麻。

**方法论**：递归自我优化系统
- **α-提示词（生成器）**：生成其他提示词或技能
- **Ω-提示词（优化器）**：优化其他提示词或技能
- 循环过程：创建初始提示词 → 用优化器优化生成器 → 用进化后的生成器创建目标提示词 → 持续迭代

**"道" - 原则**：
- 凡是 AI 能做的，就不要人工做
- 上下文是第一性要素，垃圾进，垃圾出
- 先结构，后代码，规划好框架
- 奥卡姆剃刀：如无必要，勿增代码
- 一次只做一件事

**"法" - 实践方法**：
- 一句话定义目标 + 非目标
- 能抄不写，先问 AI 有没有现成方案
- 看官方文档，喂给 AI
- 按职责拆模块，接口先行
- 一次只改一个模块

项目还包含精选提示词库、Skills 技能大全、在线提示词表格等资源，支持 20+ 种语言。


### Public APIs

> 项目地址：https://github.com/public-apis/public-apis
> ⭐ 417,206 stars

**一个由社区维护的免费公开 API 合集**，堪称开发者找 API 的宝藏仓库。

**涵盖领域**（50+ 分类）：
- 动物、动漫、艺术与设计
- 区块链、书籍、商业
- 加密货币、货币兑换
- 数据验证、开发工具
- 邮件、娱乐、金融
- 游戏、地理编码、政府
- 健康、招聘、机器学习
- 音乐、新闻、天气
- ...还有更多

**特点**：
- 完全免费，社区人工维护
- 每个 API 标注是否需要认证、是否有 HTTPS、是否开源
- 支持 Postman 一键调用
- 可直接用于个人项目或产品开发

**适用场景**：当你需要某个功能（天气、汇率、IP 定位、邮件验证等）但不想自己实现时，先来这里找找有没有现成的 API。可以让你的OpenClaw看看有没有什么有用的可以试试

## 此刻在读 · NOW

《沉思录》马可奥勒留 ，何怀宏 译

![沉思录](/assets/images/2026/meditations.png)

斯多葛哲学的核心之一：**关注你能控制的，接受你不能控制的。**

别人怎么样，他控制不了；但怎么看待别人，他能控制。所以他选择看优点、看收获，而不是盯着缺点抱怨。

为什么现代人少：

- 社交媒体放大了"吐槽"和"批评"--骂人容易获赞，夸人显得"舔"
- 竞争压力大，看别人优点反而焦虑，不如挑毛病找心理平衡
- "嘴上赢"成了习惯，忘了"心里赢"才是真的赢

马可·奥勒留的厉害之处：他是皇帝，手握大权，想杀谁、想骂谁都可以。但他选择克制、选择感恩、选择从每个人身上学东西。这不是"不敢说"，是"不必说"。



《苏格拉底的申辩》柏拉图 著

![苏格拉底的申辩](/assets/images/2026/sgld.jpg)

公元前399年，七十岁的苏格拉底站在雅典法庭上，面对"腐蚀青年"和"不敬神"的指控。柏拉图记录下这场申辩，成为西方哲学史上最重要的文献之一。

苏格拉底拒绝用华丽的修辞技巧为自己辩护，而是以惯常的问答方式，逐一驳斥指控。他说自己之所以"得罪"人，是因为他说了真话——他只知道自己一无所知。

**名场面**：当被判有罪后，苏格拉底提议"惩罚"是自己应在市政厅免费吃饭。最终，他选择饮下毒芹汁，而非流放或沉默。

这本书不长，却道尽了知识分子的命运：**说真话的人，往往不被容忍。** 两千多年过去了，雅典的法庭已经消失，但苏格拉底的精神依然在拷问每一个时代。

>《申辩》是西方最感人的作品之一，就如同《论语》一样，我们在其中能够聆听伟大的圣人最深刻的教谕："缺乏省察的生活不值得过"，士不可以不弘毅，自以为是最可笑，不能太关心外在的东西，世界上不只有生死……

也就是说苏格拉底的地位堪比中国古代的孔子，他们的共同点都是，述而不作，都是由他们的学生来回忆和记录。所以，我们有理由怀疑，这二位是不是真正存在，而不是虚构的历史人物。

https://immarcus.com/blog/why-three-dimensional-space

## 片语 · 简短的闪念

自从使用Astro，感觉良好，对于智能体现在是又爱又恨，爱的是他确实提高了人类（均线以下）的能力，恨的原因也是因为智能体的能力太强，开始逐渐抢占人类的工作位置。

所以，身在这个时代的洪流旋涡中，你无法选择。我挺佩服那些 NO AI 战士，他们在自己的领域里有着不可代替的能力，或许，某天，地球爆发了能源危机或是电子信息干扰，原能力的王者们，又会统一地球了。
