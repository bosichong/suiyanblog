## 三、安装 Steam

Ubuntu 软件中心可直接安装 Steam：

**方式一：软件中心**

打开 Ubuntu 软件（Software），搜索 "Steam"，点击安装。

**方式二：命令行**

```bash
sudo apt update
sudo apt install steam
```

安装完成后启动 Steam，登录你的账号。首次启动会下载更新，耐心等待。

## 四、Steam 兼容模式安装暗黑破坏神 4

### 4.1 下载战网安装程序

访问暴雪战网官网下载国服安装程序：

- [战网国服下载](https://www.blizzard.cn/download/)

下载得到 `BattleNet-Setup.exe` 文件。

### 4.2 Steam 添加战网安装程序

1. 打开 Steam，点击左下角「添加游戏」→「添加非 Steam 游戏」
2. 点击「浏览」，找到下载的 `BattleNet-Setup.exe`
3. 勾选该文件，点击「添加所选程序」

接下来设置兼容模式：

1. 在 Steam 库中右键点击 BattleNet-Setup
2. 选择「属性」→「兼容性」
3. 勾选「强制使用特定 Steam Play 兼容性工具」
4. 选择 **Proton 10**（推荐版本）
5. 关闭窗口，点击「播放」开始安装

安装过程中会弹出战网安装界面，按提示完成安装。安装路径保持默认即可。

### 4.3 查找战网安装目录

战网安装完成后，需要找到战网启动文件的位置。Steam 的 Proton 容器目录结构如下：

```bash
# Steam Snap 版本的路径
~/snap/steam/common/.local/share/Steam/steamapps/compatdata/
```

使用终端搜索战网目录：

```bash
find ~/snap/steam -name "Battle.net.exe" 2>/dev/null
```

或者：

```bash
# 查找 compatdata 下的所有 Proton 容器
ls ~/snap/steam/common/.local/share/Steam/steamapps/compatdata/
```

每个容器是一个数字文件夹（AppID）。找到包含 `Battle.net.exe` 的目录，记下完整路径，例如：

```
~/snap/steam/common/.local/share/Steam/steamapps/compatdata/XXXXX/pfx/drive_c/Program Files (x86)/Battle.net/Battle.net.exe
```

### 4.4 新建 Steam 游戏指向战网

现在需要把战网本身添加为 Steam 游戏：

![Steam 游戏指向战网](https://www.suiyan.cc/assets/images/2026/d401.png)

1. Steam 左下角「添加游戏」→「添加非 Steam 游戏」
2. 「浏览」找到刚才记录的 `Battle.net.exe`
3. 勾选并添加



设置兼容模式：

![Steam 游戏指向战网](https://www.suiyan.cc/assets/images/2026/d402.png)

1. 右键 Battle.net →「属性」→「兼容性」
2. 强制使用 **Proton 10**
3. 「目标」字段填写完整路径
4. 「起始目录」填写 exe 所在的目录

完成后点击「播放」，战网应该能正常启动。如果启动失败，尝试更换 Proton 版本（Proton-GE 或 Proton Experimental）。

## 五、国服反和谐设置

国服版本对部分内容进行了本地化处理，包括 CG 动画、怪物外观、特效等。如果想体验原版画面，需要进行反和谐操作。

### 5.1 创建配置文件

找到暗黑破坏神 4 的游戏安装目录（在战网的 Proton 容器内），创建配置文件：

**步骤一：创建 WTF 文件夹**

在游戏目录下创建 `WTF` 文件夹：

```bash
# 游戏目录示例
~/snap/steam/common/.local/share/Steam/steamapps/compatdata/APPID/pfx/drive_c/Program Files (x86)/Diablo IV/
```

在游戏根目录（包含 `Diablo IV.exe` 的目录）下创建 `WTF` 文件夹。

**步骤二：创建 Config.wtf 文件**

在 `WTF` 文件夹中创建文本文件，重命名为 `Config.wtf`（注意扩展名）。

**步骤三：写入配置**

用文本编辑器打开 `Config.wtf`，写入以下内容：

```
SET OverrideArchive "0"
```

保存并关闭。

### 5.2 添加启动参数

在战网客户端中添加启动参数：

1. 打开战网客户端
2. 点击暗黑破坏神 4 的齿轮图标
3. 选择「游戏设置」
4. 勾选「额外命令行参数」
5. 输入：` -enableagentmanager`（注意 `-` 前面有一个空格）
6. 点击「完成」

重新启动游戏即可生效。

**注意**：反和谐操作依赖本地文件，如果游戏更新可能导致失效，需要重新检查文件。

## 六、常见问题与注意事项

### Proton 版本选择

- **Proton 10**：推荐使用，兼容性最佳
- **Proton-GE**：社区增强版，部分游戏性能更好
- **Proton Experimental**：最新功能，可能不稳定

如果遇到问题，尝试切换不同版本。

### 显卡驱动

NVIDIA 用户务必安装官方驱动：

```bash
# 检查当前驱动
nvidia-smi

# Ubuntu 驱动管理
sudo ubuntu-drivers devices
sudo ubuntu-drivers autoinstall
```

AMD 用户推荐使用 Mesa 开源驱动，Ubuntu 默认已安装。

### 游戏存档位置

存档位于 Proton 容器内：

```bash
~/snap/steam/common/.local/share/Steam/steamapps/compatdata/APPID/pfx/drive_c/Users/用户名/Saved Games/Blizzard/Diablo IV/
```

### 常见启动失败原因

1. 显卡驱动版本过低 → 更新驱动
2. Proton 版本不兼容 → 切换版本
3. 缺少依赖库 → 安装 `libvulkan1` 等库
4. 战网登录异常 → 检查网络连接

### 性能优化建议

1. 游戏内设置适当降低画质
2. 关闭不必要的后台程序
3. 使用 SSD 存储
4. 确保系统散热正常
