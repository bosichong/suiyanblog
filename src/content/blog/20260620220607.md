## PNG 元素分离工具

将透明背景 PNG 中每个独立元素自动分离为单独的 PNG 文件。

![截图](/assets/images/2026/split_image.png)

[github 仓库地址](https://github.com/bosichong/split_image)



使用方法
## 安装依赖
pip install -r requirements.txt

## 启动
python split_png.py
点击「选择图片」，选取透明 PNG 后，程序会在同目录下创建 图片名/ 文件夹，保存 element_0.png、element_1.png 等文件。
