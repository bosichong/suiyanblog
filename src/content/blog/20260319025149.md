自从用了 OpenClaw 之后，很多计算机和网络方面的操作都发生了巨大的改变。以前很多需要亲力亲为的事情，现在都可以交给"龙虾"处理了。

比如说现在这篇博文，我是使用豆包输入法在 iPhone 的备忘录里录入的。之后我会交给龙虾，让它帮我进行简单的文字、标点及语句通顺处理，不进行润色和扩写。然后让它创建一个新的博客文章，填入标题、标签和简介。确定没有问题之后，直接推送到 GitHub。GitHub 再同步到 Vercel，服务器进行编译，把静态资源部署到服务器，页面就更新了。我还可以让龙虾把最新的博客同步到 Mastodon 发布一篇嘟文，甚至发布到推特 X。

当然，如果你使用 WordPress 这个老牌博客程序的话，也有相关的龙虾技能。WordPress 应该也支持 API 操作，这样你也可以用龙虾进行相关的博客操作。还有一些常见的个人博客程序，基本上都能找到相关的龙虾技能，有兴趣的可以试一试。

有的人可能会说，这样的话独立博客就失去了它的意义。但是养龙虾到底是为了什么？我觉得 OpenClaw 改变的不仅是操作，更是人的思想。OpenClaw 并不是代替人类来做这些事，它只是在帮助我们提高效率。而做这些事之前，人类都需要进行一些思考。如果你只给它一个标题，让它写一篇文章，就算写一万篇也没有任何意义——因为人类不需要思考的话，用不了多长时间，这个人就废了。

用龙虾的意义，和用豆包语音输入法差不多，都是为了节约时间、提高效率。这样我们就能省下更多的时间去做其他事情：看书、学习更多技能、陪家人，甚至遛狗、烧烤、喝酒，这不是挺好吗？我们所做的一切不都是为了让自己有更多自由的时间吗？难道你不想节约更多的时间成本、创造更多的价值吗？

我觉得这是一个值得思考的问题，这也涉及到哲学，这也是一个趋势。之前没赶上初代的大模型聊天，也没赶上二代的辅助编程，那么到了龙虾的时代，你还在等什么？赶紧上车吧！
