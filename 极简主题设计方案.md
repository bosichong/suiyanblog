# 极简主题设计方案

## 一、设计目标

打造一个极致简约、现代感强的中文博客主题，以"做减法"为核心设计理念，专注于内容阅读体验。

## 二、设计原则

1. **极简主义**：去除所有非必要元素和动画效果
2. **移动优先**：以移动端体验为主，不做响应式适配
3. **单主题**：只保留浅色主题，移除主题切换功能
4. **黑白灰配色**：以黑灰为主色调，营造现代感
5. **静态优先**：所有图标使用静态 SVG，无动态效果

## 三、配色方案

### 主色调系统
- **背景色**：`#F8F8F8`（比纯白稍灰，减少视觉疲劳）
- **内容背景**：`#FFFFFF`（纯白，突出内容）
- **主文字**：`#1A1A1A`（近黑色，高对比度）
- **次要文字**：`#666666`（中灰色）
- **辅助文字**：`#999999`（浅灰色）
- **边框/分隔线**：`#E5E5E5`（浅灰色）
- **链接文字**：`#333333`（深灰，hover 时变为 `#1A1A1A`）
- **链接下划线**：`#CCCCCC`（浅灰，增强可识别性）

- ~~金色主题色 `#F7B750`~~
- ~~暗色主题全部配色~~

## 五、布局结构重构

### 整体布局
```
┌─────────────────────────┐
│  [LOGO] | [导航菜单]     │ ← 固定顶部导航
├─────────────────────────┤
│                         │
│     博客内容区域         │ ← 最大宽度 700px
│                         │
├─────────────────────────┤
│    SNS 图标 | 版权信息   │ ← 底部
└─────────────────────────┘
```

### 导航设计（顶部固定）
- **样式**：单行横向排列，背景 `#F8F8F8`，底部 1px 边框，高度 56px
- **Logo**：左侧，显示博客名称 "碎言"，字重 600，字号 16px，黑色
- **菜单项**：右侧，用 `/` 分隔
- **菜单项**：归档 / 标签 / 搜索 / 友链 / 关于
- **字体**：14px，深灰色 `#333333`，字重 400
- **分隔符**：`/` 斜杠，颜色 `#CCCCCC`，左右各 8px 间距
- **Hover**：文字变黑，颜色 `#1A1A1A`
- **布局**：flex 布局，justify-content: space-between

### 内容区域
- **最大宽度**：700px
- **左右内边距**：16px（移动端）
- **顶部内边距**：80px（为固定导航留出空间）
- **字体大小**：16px（正文）
- **行高**：1.6
- **颜色**：`#1A1A1A`

### 底部设计
- **SNS 图标**：横向排列，图标大小 20px，颜色 `#666666`
- **版权信息**：居中，12px，浅灰色 `#999999`

## 十四、注意事项

1. 移动端固定导航需要预留顶部空间
2. 所有链接保持可访问性（下划线或颜色对比）
3. 图标 SVG 需要确保 viewBox 正确
4. 移除动画后，交互反馈通过 hover 样式体现
5. 确保所有文字在白色背景上有足够对比度

## 十五、现代感极简细节设计

### 1. 微妙的间距与留白
- **页面外边距**：左右保持足够的留白（16px），让内容呼吸
- **段落间距**：正文段落之间 24px 间距，提升阅读舒适度
- **列表项间距**：16px 垂直间距，保持视觉连贯性
- **区块间距**：不同内容区块之间 32px 间距，清晰分隔

### 2. 精致的线条处理
- **细边框**：使用 1px 细边框，颜色 `#E5E5E5`，极简而精致
- **分隔线**：使用 `#E5E5E5` 的 1px 实线，不添加阴影
- **下划线**：链接 hover 时添加 1px 下划线，颜色 `#CCCCCC`
- **线条一致性**：全站统一使用 1px 线条，保持视觉统一

### 3. 极简的排版细节
- **行高**：正文 1.6，标题 1.3，确保阅读舒适
- **字间距**：使用默认字间距，不刻意调整
- **对齐方式**：左对齐为主，保持自然阅读流
- **首行缩进**：中文段落首行缩进 2em，符合阅读习惯

### 4. 色彩的层次感
- **主色**：黑色 `#1A1A1A` 用于标题和重要内容
- **次色**：中灰 `#666666` 用于正文和次要信息
- **辅助色**：浅灰 `#999999` 用于辅助文字和元信息
- **背景层次**：`#F8F8F8` 页面背景，`#FFFFFF` 内容背景，形成清晰层次

### 5. 交互的微妙反馈
- **链接 hover**：颜色加深 + 1px 下划线，简洁明了
- **按钮 hover**：轻微背景色变化（透明度 0.05）
- **图标 hover**：颜色从 `#666666` 变为 `#1A1A1A`
- **无动画**：所有交互即时响应，无延迟和过渡

### 6. 极简的装饰元素
- **无装饰**：去除所有装饰性图形、图案、纹理
- **纯色块**：仅使用纯色背景，无渐变、无纹理
- **简单几何**：图标使用简单的几何线条，无复杂细节
- **留白即设计**：通过留白和间距创造视觉美感

### 7. 现代感的字体处理
- **字重对比**：标题 600 vs 正文 400，形成适度对比
- **字号比例**：采用 1.2 黄金比例（16px → 19.2px → 23px → 28px）
- **字体家族**：系统字体栈，无额外字体加载
- **中英文混排**：中文字体优先，英文使用系统默认

### 8. 统一的视觉语言
- **圆角**：全站统一使用 4px 小圆角（仅在必要时）
- **阴影**：完全移除阴影效果，扁平化设计
- **渐变**：完全移除渐变效果，纯色设计
- **边框**：统一 1px 边框，颜色 `#E5E5E5`

### 9. 内容的视觉层次
- **标题突出**：使用更大字号和更黑颜色
- **信息分层**：通过字号、颜色、位置区分信息层级
- **重点强调**：仅通过颜色和字重，不使用特殊样式
- **元信息弱化**：日期、标签等使用浅灰色，不抢眼

### 10. 移动端适配细节
- **触摸区域**：最小 44x44px，确保可点击性
- **字号适配**：移动端不缩放，保持 16px 基准字号
- **间距调整**：移动端保持 16px 间距，不压缩
- **导航固定**：顶部固定导航，高度 56px，预留空间

## 十六、设计特色

- **极简配色**：黑白灰三色体系，视觉纯净
- **无干扰阅读**：去除所有特效，专注内容
- **现代排版**：合理的字号、行高、间距
- **移动优先**：针对移动端优化的布局和字号
- **轻量快速**：无动画、少依赖、加载快
- **细节精致**：通过间距、线条、色彩层次营造现代感
