Ubuntu 24.04 如何运行《原神》？问了kimi尽然给出官方有Linux版本的原神答案，吓得我赶紧问了问Gemini。

### 使用“An Anime Game Launcher”

这是目前 Linux 社区最成熟、最自动化的方案，专门为米哈游旗下的游戏（原神、星铁、绝区零）优化。

1. **安装 Flatpak 环境**（如果还没装）：
```bash
sudo apt update
sudo apt install flatpak
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo

```


2. **安装 Launcher**：
在应用商店搜索 **"An Anime Game Launcher"** 或者通过命令行安装：
```bash
flatpak install flathub moe.launcher.an-anime-game-launcher

```


安装完Launcher后，记得重启一下机器，不然`An Anime Game Launcher`在应用菜单里是不会显示的，重启后打开，进行下一步。

重启后打开应用：

![An Anime Game Launcher](https://suiyan.cc/assets/images/2025/ys01.png)

3. **运行与配置**：
* 启动后，软件会自动帮你处理 Wine 环境和各种补丁。
* 你只需要按照提示选择安装路径下载游戏即可。
* **优点**：一键式操作，支持 FPS 解锁、自动配置环境。


更新完补丁后就可以下载游戏了：

![原神下载登录器](https://suiyan.cc/assets/images/2025/ys02.png)

### ⚠️ 重要提示与注意事项

1. **显卡驱动**：
* **NVIDIA 用户**：必须安装专有驱动（通过“软件和更新”中的“附加驱动”安装最新的 550+ 版本）。
* **AMD 用户**：Ubuntu 24.04 自带的 Mesa 驱动通常表现良好。


2. **反作弊风险**：虽然目前大部分 Linux 玩家都能正常游玩且未被大规模封号，但**这仍属于非官方支持的行为**。建议不要在 Linux 端使用任何第三方修改插件（如脚本、宏）。
3. **文件系统**：请务必将游戏安装在 Linux 原生格式的分区（如 `ext4`），**不要**安装在 NTFS（Windows）分区上，否则会出现权限错误或启动失败。
4. **网络报错 (10010/4001)**：如果启动时遇到网络断开报错，可以尝试在进入游戏加载界面时暂时关闭网络，待进入后再开启（部分版本的临时绕过方法）。
