作为一个东北人，我都不好意思说，今天终于下了一场入冬以来真正意义的「雪」！



这雪下得这是够着急的，你不赶紧拿手机拍几张照片，估计一会就会融化了。磨磨唧唧唠唠叨叨念叨了一个冬天的雪，今天终于下了！然后并没有什么特别的感觉，可能是因为某些事，你满怀期待的等待，最终所有的激情都被长久的时间无情的磨灭了。我甚至连放张图在博客里的欲望都没有了，或许是懒得。




这个博客没有什么花销，就是一个域名的钱，昨天把费用续上了，同时，也续费了微信读书的VIP。

有的时候，我是不是有点抠门了，你看人家对待自己的博客（曾经浏览过一个博客），光博客的主题就花费大几百，更别说什么主机、cdn、防火墙等等，一套下来得个千八的。我看他的博客也没啥流量，动不动还被攻击，是不是就打不开了或是重新开始......不理解，反正看过一次就没再去第二次。反观自己，我就是觉得自己的博客无非就是一个记事本，把自己的平日里的一些记录分享到网上而已。就算那天打不开了，那些.md还是在Github和本地保存着，至于又没人看，一切随缘。

另外一个爱好就是折腾主题，不过今年开始，我一直在做减法，前几天我把tailwind给砍掉了，现在的样式就是基于`pico.css`。


> 极简CSS框架 语义化 HTML 一个极简轻量级的入门套件，优先考虑语义语法，默认情况下使每个 HTML 元素都具有响应性和优雅性。编写 HTML 代码，添加 Pico CSS 样式，大功告成！


我想以更简洁的方式来修改这个博客的主题，在我的一顿砍的优化下PageSpeed Insights（PSI）终于拿到了移动端和PC端的 双400分的满分，也就是说，在用户浏览体验这方面，我做的绝对是有诚意的！


![PageSpeed Insights（PSI） 满分 ](https://suiyan.cc/assets/images/2025/26e3801a-46ad-4125-966f-68a0674f6a3e.png)

![PageSpeed Insights（PSI） 满分 ](https://suiyan.cc/assets/images/2025/18246552-23e9-4f02-a5d8-7ab8552a19f0.png)


这次 我直接使用了一个默认色调的主题，可以说是没做任何的修改，我觉得任何人可能都会修改主题，就像使用wordpress，很少有人使用官方默认的主题，但是默认主题其实都是官方优化的很好的主题，用户体验是绝佳的，大部分博主都需要个性的展示来弥补博客本身内容不足或是质量不行的缺点......看到这里有的朋友可能会要骂人了，我也相信一个好看的博客，并且有绝佳的内容，这就像一个漂亮的美女，发出动人的声音，确实很招人喜爱。但是，我们只是一个普通人，美女帅哥们都在拍短视频，醒醒吧。

本来没啥人看，我感觉我还会变得越来越让人讨厌，说到讨厌，推荐看看老莫的新博文：[我为什么招人讨厌？](https://onojyun.com/2026/03/03/%e6%88%91%e4%b8%ba%e4%bb%80%e4%b9%88%e6%8b%9b%e4%ba%ba%e8%ae%a8%e5%8e%8c%ef%bc%9f/)。我就觉得博客首先是要有自己的个性，但是有了个性就会有冲突。我不是那种每个博客必去留言，有访客必回访的操作，这种类似于无效社交的行为以前我也做，现在我只看我喜欢的。

> 如果你要了解一个博客，就去看看他的友情链接，他不会放上自己不喜欢的博客。

我承认，这是我之前放到友情链接里的，我删除了后半段他不会放上自己不喜欢的博客。我的博客目前有一些是之前主动去申请互换的所谓的**友情链接**，可是我发现，我和这些博主之间并没有什么友情，甚至还不如陌生人亲切，当初可能是觉得有个友情链接，没有链接在上边太磕碜了吧，所以就厚着脸皮去申请，事到如今，我觉得还没把我删除的博主是很有道义的，至少他们遵守了互利的原则（我为了那点零星的流量互换），我也暂时不打算删除了，至少哪些可以证实我当时无知的操作，告诉我以后不要在那么做了。


啊，还有个微信读书VIP，可能有很多人不理解，为啥要冲这个VIP？不充值也能看，但是每天要打卡、签到、参加一些强制活动，才能有机会得到更多的书。这些操作直接影响了我的阅读体验，本着每年也会买很多纸质书，细算充值VIP也是很划算，没了干扰看书更方便。
