> 我一直认为，能否极简地生活，要看你能否辨别出什么才是真正重要的东西。能够分辨什么是重要之物，就是一种智慧，就是明禅理。—《禅与极简生活艺术》

## 禅与极简生活艺术

最近迷上了极简，突然觉得极简是一种智慧。至于为什么开始极简，生活、工作、学习，甚至是博客？我在进行这些极简操作的同时，也一直在考虑这个问题。前几天看了一本书：《禅与极简生活艺术》。我对这本书的推荐程度是一般吧，因为这本书的作者是一位日本禅修大师，他在书中借鉴了大量的佛家禅修事例，而我只是想了解极简生活和极简主义，可能是我并不理解把禅和极简合并到一起的精华吧。当然，这本书当中有一段可能是作者没有想到的，正是因为这一段，可能会有很多人对这本书做出一般或者很差的评价。他在书中写道：

>前几天我坐了一次电车。当时，在我前面的座位上，坐着一个年轻的女孩。这个女孩身穿时下流行的漂亮服装，外貌看起来也很漂亮。从周围的男乘客都时不时地看她就可以看出，她确实很漂亮。然而，这个女孩刚一坐下就从手提包中取出化妆工具，对着小镜子化妆。看到她做出这番举止后，周围人都不约而同地将视线从她身上移开。毫无疑问，我也情不自禁地闭上了眼睛。


其实在前边读的时候，一直觉得这本书还是很不错的，直到我看到上面这段文字，之前所有不错的感觉都被这段话给抹杀了。可能我还没有达到作者高僧的那个境界吧，而且十分不理解他这么写的含义。然后就是勉强把这本书读完，最后给了一个一般的评价。

但是这本书里关于极简的阐述还是很不错的，他介绍了大量关于生活、工作、情绪以及人际关系中的极简之道，这些极简的理念还是值得我们去学习的。那么什么是极简？

## 什么是极简？

>简单可能比复杂更难：你必须努力理清思路，让它变得简单。但这一切都是值得的，因为一旦你达到了那种简单，你就能移山。——史蒂夫·乔布斯

**少即是多，生活的艺术在于持续地摆脱多余的东西。**

我所理解的极简：

* 极简生活:专注内心真实的需求，减少无效社交，家人朋友永远放在第一位。
* 极简工作:专注核心任务，排除干扰。
* 数字极简:电脑和手机，删除无用的软件，定期深度整理数字内容。
* 极简博客:功能极简、样式极简、内容极简。

当然，极简不是越少越好，是刚刚好，是一种平衡，或者说可以少到让你不必分心或过度关注。

## jQuery的极简主义哲学

2006年，John Resig发布jQuery时提出的口号"Write less, do more"，不仅是营销话术，更是一套完整的软件极简主义哲学。

jQuery大家应该都接触过，尤其是那些早期在互联网上折腾网站的站长，对于jQuery的使用应该是很多的。早些年，我在网页端制作一些交互操作的时候，使用原生的JS写是很复杂的，至少对于我来说是很复杂、很有难度的。有一段时间我接触到jQuery，不太相信代码的编写可以这样的简单和优雅——编写更少的代码，做更多的事情，这不就是我们想要的吗？

## 践行极简

### 极简博客

这阵子我可以说是正在大刀阔斧地执行极简主义的理念和生活方式。首当其冲的就是把博客的源代码和主题进行了极为大胆的精简，无论是从编写方式还是从主题设计，都采用了极简开发和极简设计原则。对于博客的编写，我删除了大量无用的功能，这样就减少了大量的依赖，使当前博客整体的代码量大幅度减少。

博客主题也采用了浅色界面、大量留白的极简风格，删除了所有动画以及一些多余的功能，哪怕只要让我感觉有一点多余，我都要给它删除。你现在看到的界面就是我对博客进行极简优化的结果。由于结构的精简代码的减少，我对于细节的掌控更加细致入微了。我删除了站内的所有的自定义css，博客的所有样式的细节全部采用tailwind CSS框架的class

>Tailwind CSS 的工作原理是扫描所有 HTML 文件、JavaScript 组件以及任何 模板中的 CSS 类（class）名，然后生成相应的样式代码并写入 到一个静态 CSS 文件中。
> 他快速、灵活、可靠，没有运行时负担。

对于博客的一些数据统计，我通通都删除了，包括文章统计、访问量统计，我都给删除了。因为我不想看到这些数据，我构建这个博客、撰写这些博文，并不是因为这些数据的多少才去创作。我去编写代码撰写文字，是因为我想做我喜欢，而不是因为看的人多了我才去做。**彻底消除数字焦虑，无论是我还是读者。**

删除访问统计，不留下任何人的访问信息，尊重来访者的隐私。而且我把文章的评论设置为默认隐藏，我想读者是因为他想评论而去评论，而不是因为看到很多人在评论而去评论。

删除了太多无用的HTML代码结构，优化了页面的打开速度。网站的ping值只是一个单纯的测试网站是否能打开的连接速率的侧面数值。对于用户来说，网站的体验的多方面的，这个可以参考Google PageSpeed Insights [https://pagespeed.web.dev/](https://pagespeed.web.dev/),

> PageSpeed Insights (PSI) 可提供有关网页在移动设备和桌面设备上的用户体验的报告 并提供有关如何改进该页面的建议。

PSI是专业可以信赖的，可以作为极简网站改进的依据。


### 极简生活

断离舍的极简生活并不是随意的扔东西，有价值的东西还是要保留的，极简空间的精髓应该是去除杂物，这好比去除心中的杂念一般了。临近新年，正是大扫除的时候，趁机可以清楚一些旧物，例如很多的旧书，我的书架上有很多老旧的编程书籍，大约能有二三十本，是时候该处理更换一些新的书籍了。不过，最近喜欢使用微信读书来阅读，功能虽然不多但胜在实用，这也是用户对软件极简设计的反馈吧。

最近，我还把车里的杂物清理了干净了。以及更换了一个装杂物的箱子，因为之前的太大了。我非常反感车里堆满各种东西，后背箱塞得满满的。车子就是载人和装载必要的东西，其他的真是没必要放在车里了。

目前我还是极简生活的初级阶段，很多极简的生活方式还有待于我去体验。

### 极简人际关系

人际关系有的时候会让人很疲惫的，其实人际关系并不需要刻意的去精简，当你理解了人与人交往的目的，你自然就会产生一个极简的人际关系。当然，如果你有能力处理复杂的人际关系，另当别论了。


## 误区

当然，过度的极简也是一种错误，例如为了极简而极简，是自己陷入了一个新的焦虑。为了物品上的极简，过度的丢弃具有价值和意义的物品，这都将是很重要的损失。也不要为了达到某些极简行动的数值而进入强迫的极简操作行为，这些都是极简主义践行的误区。


## 总结：极简主义的本质

极简主义不是：
* ❌ 苦行僧式的生活
* ❌ 昂贵的"极简风"装修
* ❌ 统一的标准（如必须少于100件物品）
* 
极简主义是：
* ✅ 为生命做优先级排序的智慧
* ✅ 从占有者变为使用者的视角转换
* ✅ 认清什么对自己最重要后，有勇气放弃其他的选择力
正如日本禅宗所言："当你意识到无一物欠缺时，整个世界都属于你

极简是一种智慧，也是一种艺术，正如日本禅宗所言：**当你意识到无一物欠缺时，整个世界都属于你。**。
