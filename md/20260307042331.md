RSS刷到老莫的新文：[我为什么关闭博客评论？](https://onojyun.com/2026/03/07/why-i-closed-my-blog-comments/)，文章侃侃而谈，从为何要关闭博客评论、转而投向邮件和Telegram去追求高质量的互动，一直讨论到独立博客的「独立」与「博客」。借此我也说说我对独立博客的一些想法。

## 万维网的设计哲学

万维网（World Wide Web）的设计哲学深刻影响了现代互联网的发展，其核心思想围绕着开放性、去中心化、简单性和普适性展开。

为什么要提到万维网？如果没有万维网，就没有博客以及任何可以在网络上发表文字和图片的基础。所以，无论是什么页面，只要能在互联网打开的，基本上都离不开这个最基本的设计哲学。独立博客自然也包括这些基本性质，例如：独立域名（去中心化）、任何人可以查阅（开放性）、任何终端只要可以打开浏览器、能够联网就可以浏览博客（简单和普适性）。可以说这是万维网的基本性质，也是每个独立博客的根本。blog（博客）大概始于1997年，2000年以后兴起，国内大约晚几年。blog中的log是日志的意思，其含义可以理解为记录、日记等概念。后续个人独立博客大放异彩，学术、艺术、文学、计算机领域涌现出无数知名大佬的独立博客。

独立，我理解为去中心化。因为你有一个独立的域名，一个独立的个体在维护、撰写这个博客，这个博客有着你独立的思想、风格和表达欲望。独立博客的到访者大多来自搜索引擎、RSS、聚合网站、友情链接以及个人转发推荐等。与这些相对的就是平台化的写作，例如知乎、微博、X、YouTube等大型平台，他们会把我们发出的信息聚合，通过大数据筛选推送给终端。当然，某些情况下，基于国情的某些限制，*你可能有很多话不能说。*

这些都是独立博客的前提，之后才到了独立博客的个性。因为是独立博客，所以博主就是这个博客的"绝对领导者"，那么每个博客就会有着自己的使命，这个使命就是博主的表达出口。如果有一定计算机和编程基础，都会了解到网络的基础——**TCP/IP协议**，它有一个很有意思的**握手**：

```
客户端                    服务器
|    ① SYN (我能连你吗？)    |
| ------------------------> |
|  ② SYN-ACK (能，你呢？)   |
| <------------------------ |
|    ③ ACK (我也能，开始吧)  |
| ------------------------> |
|        连接建立 ✓          |
```

有的时候我就在想，博客的评论大概就是这样的吧：

1. 发表博文
2. 发表评论
3. 回应评论
4. 建立或失去连接（赞同或反对，喜欢或讨厌）

赞同和反对不代表会失去连接，但是讨厌几乎可以断定会失去连接，甚至有时候会在看到评论的时候没有回应，直接失去连接。

## 无效社交

对于只进行到发表评论和回应评论的交往，我想这就是平日里所说的**无效社交**吧。当然，有沉迷于评论数目的，也有沉迷于评论质量的，这都是每个人的内心所向。确实有一部分人——应该说是很大一部分人——只关注流量和评论数。商业目的的博客不在此讨论范围，因为那已经不再是独立博客范畴了。平心而论，你喜欢打招呼类型的评论还是走心的评论？先不要急着回答，因为这没有绝对的答案，这个问题本身就是一个带有其他隐藏附加条件的问题。若是美女博主发了一组自拍的美图，她希望每个人都是走心的评论，还是更多的夸她的评论？如果一个博主受了委屈或是经历了一些事情，他需要的是更多安慰评论的情绪价值，还是客观的走心批评？当然像老莫那种高质量博主，我不敢妄加评论，但是他需要的回应已经不仅仅只是"走心评论"能代表的了。

## 独立博客的本质

不管怎么说，每个独立博客都会有我之前说的万维网的基本性质（除了上密码限制浏览、收费浏览、拒绝搜索引擎收录等以其他限制的不在讨论范围内）。你发出的东西就是开放的、给其他「终端」看的，哪怕你说是给自己写的。在此基础之上才是独立博客个性的开始。

至于老莫，

>如果你觉得被我冒犯了，我向您诚挚地道歉。

我觉得这句话就他妈多余了。因为独立博客有自己的自由，浏览者也一样有不选择去浏览你的博客的自由，双向的自由，没有谁对不起谁。哪怕就是你在骂我，我也去看，我就是贱就是喜欢，这才是自由。
