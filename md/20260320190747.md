装好 OpenClaw 只是第一步，真正让它干活的是 Skills。

就像 VS Code 装插件、Chrome 装扩展，OpenClaw 的能力边界由你安装的技能决定。折腾了一圈，这 10 个技能我认为是必装的。

## 搜索类：让 Agent 连接真实世界

### 1. tavily-search

专为 AI 设计的搜索引擎，返回结构化结果，不用自己解析网页。

| 功能 | 说明 |
|------|------|
| 搜索 | 网页搜索，支持域名过滤 |
| 提取 | 获取 URL 内容，转 Markdown |
| 爬取 | 深度爬取网站 |
| 映射 | 发现站点 URL 结构 |

安装后配置 `TAVILY_API_KEY`，免费额度每月 1000 次，日常够用。

### 2. multi-search-engine

17 个搜索引擎整合，最大亮点是**无需 API Key**。

| 国内引擎 | 国际引擎 |
|----------|----------|
| 百度、360、搜狗 | Google、DuckDuckGo |
| 微信、头条 | Brave、Startpage |
| 必应（中/英） | WolframAlpha |

支持高级搜索语法：`site:`、`filetype:`、时间过滤。隐私党可以直接用 DuckDuckGo。

## 内容处理：从混乱到有序

### 3. summarize

一个命令摘要网页、PDF、YouTube 视频：

```bash
summarize "https://example.com" --model google/gemini-3-flash-preview
summarize "/path/to/file.pdf"
summarize "https://youtu.be/xxx" --youtube auto
```

支持调整长度：`--length short|medium|long`。阅读长文、看视频前先摘要，效率翻倍。

### 4. markdown-converter

把任意文档转成 Markdown，方便 LLM 处理：

```bash
uvx markitdown report.pdf -o report.md
uvx markitdown slides.pptx > slides.md
```

支持格式：PDF、Word、PPT、Excel、HTML、CSV、JSON、图片（带 OCR）、音频（带转录）。

### 5. news-aggregator-skill

8 大信息源聚合，一键获取热点：

| 来源 | 内容 |
|------|------|
| Hacker News | 技术圈热点 |
| GitHub Trending | 开源趋势 |
| Product Hunt | 新产品 |
| 36Kr、腾讯新闻 | 国内科技财经 |
| V2EX、微博 | 社区讨论 |

支持关键词过滤、深度分析，生成简报风格的报告。

### 6. rss-subscribe

订阅任意 RSS 源，获取最新文章：

```bash
python3 scripts/rss.py <rss_url> --limit 10
```

适合跟踪博客、新闻网站的更新，不用一个个打开看。

## 效率工具：干掉重复劳动

### 7. openclaw-github-assistant

在聊天里操作 GitHub：列仓库、查 CI、创建 Issue。

需要配置 `GITHUB_TOKEN` 和 `GITHUB_USERNAME`，之后就能用自然语言管理项目。

### 8. email-sender-tw

邮件发送工具，支持模板、附件、批量发送。

配置 SMTP 后，一行代码发邮件：

```python
send_email(to="xxx@qq.com", subject="标题", body="内容")
```

适合自动化通知、日报周报场景。

### 9. docker

Docker 操作指南，涵盖镜像构建、Compose 编排、网络配置、安全加固。

关键规则：
- 镜像版本要固定，不用 `latest`
- RUN 命令合并，减少层数
- 容器内不用 root
- 设置资源限制，防 OOM

### 10. humanizer-zh

去除 AI 写作痕迹，让文字更自然。

基于维基百科的"AI 写作特征"指南，检测并修复以下模式：
- 夸大的象征意义
- 宣传性语言
- 三段式法则
- AI 高频词汇

写公众号、发文章前跑一遍，味道更正。
