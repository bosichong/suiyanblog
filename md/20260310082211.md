十天，足够一些事情发生与改变……

例如：OpenAI前几天刚刚发布的GPT-5.4，不仅是"更会聊天的AI"，而是**首个能真正操控你电脑的智能体**。它把编程、推理、视觉理解融为一体，在44种专业职业任务中，83%的表现达到或超越了人类专家水平。**就问你怕不怕？**

上个月，我犹豫不决的预谋开启一个周期性的文字记录的准备，从模式上来看这很像「周刊」，但实际更多是我自己近十天内的一些见闻和解读。Vol.1 · 起点，就是在这种踌躇不决的心情下起笔了。

很开心在我的「周刊」开张之际得到了很多人的支持，老莫（莫比乌斯）还曾专门为“我”写了一篇“序”：《新周刊劝退指南》由于没有得到授权，暂时也无法在此透漏此文的详细内容。「十日后」没什么大不了的，你就当做在浏览一个普通的博客文章就好，如果不巧你正闲着无聊看到此处，喜欢或者讨厌，你都憋着，反正你看完了也不会跑去底下留言。

###  localsend

无需云端即可分享文件。  
快速、私密、离线。
面向所有人的开源跨平台文件共享。

https://localsend.org/zh-CN

使用感受：

因为更换了系统，目前正在使用ubuntu，所以在不同平台之间的文件互传成了一个不痛不痒的小问题，那么有些同学肯定会说，你可以使用QQ、微信啊。这不符合软件设计的单一职责，我希望这个应用他只负责本地局域网中文件的分享和传输，而不是我要去超市只能坐飞机去，使用飞机的运输功能来实现这个到达目的地的需求。 localsend 使用方法简单，上手快，你只要跟着你的需求进行操作就好，不用去刻意的学习如何操作，而且使用完，你只要关闭他就行，没有注册和登录，任何功能都完美的封装，你只需要进行你的文件分享传输的操作即可。

## 此刻在读 · NOW

### 在读书籍

**《苏菲的世界》**

《苏菲的世界》（_Sophie's World_）是挪威作家**乔斯坦·贾德**（Jostein Gaarder）于1991年创作的哲学启蒙小说，被誉为**20世纪百部经典著作之一**

经典名句：
    
    - "唯有清晰地意识到有一天她终将死去，她才能够体会活在世上是多么美好"
        
    - "亲爱的苏菲，我不希望你长大之后也会成为一个把这世界视为理所当然的人"

小说按时间顺序梳理了西方哲学的主要发展阶段：

| 时期              | 代表人物               |
| --------------- | ------------------ |
| **古希腊哲学**       | 泰勒斯、苏格拉底、柏拉图、亚里士多德 |
| **中世纪哲学**       | 奥古斯丁、托马斯·阿奎那       |
| **文艺复兴/巴洛克**    | 笛卡尔、斯宾诺莎、洛克        |
| **启蒙运动**        | 休谟、康德              |
| **浪漫主义/德国唯心主义** | 黑格尔、祁克果            |
| **现代哲学**        | 马克思、达尔文、弗洛伊德、萨特    |
——以上为kimi整理本书相关的简介。

说说我的感受，在读此书之前，我对哲学的认知进存在于知道「哲学」这二个字而已，我并不知道哲学存在的意义以及哲学的历史，如果你打算开始了解哲学，那么就是在翻看古老悠久的哲学历史。「你是谁」「世界从何而来」翻看此书之后你会不断地改变对自己对世界的看法，你会初步的了解西方哲学家和西方哲学的历史。本是很枯燥的哲学内容却以生动的小说形式讲述给一位14岁的女孩来听。目前我读到理性主义和经验主义文艺复兴，此处讲述了洛克、休谟、康德等经验主义哲学家和他们自己的哲学理论。

其中，经验主义对我来说比较更接近一些，毕竟这是一个靠感官来实现的哲学观点，大多数普通人可能或多或少会觉得看得见摸得着更加真实。相比古希腊的苏格拉底、柏拉图等一些哲学家的理性理论，我觉得普通人并不见得就是很理解，更多的是拿这些奇怪的哲学家名字和哲学名词装装高大上的样子吧。

我不算剧透书中的具体情节，因为这是一本需要你去读的书，大模型当下的世界，阅读开始变得珍贵，如果你打算让大模型总结这本书，那么你会错过很多本书对你思想上的冲击。

昨天，《苏菲的世界》也看完了，结尾真的是和想象中大不一样，浪漫主义+超现实主义+童话般的惊奇的结尾。15岁的小孩们滚在一起，稍后女孩告诉妈妈说我要生了，我貌似就记得这段了。说好的看书学习哲学史呢。合上书，我也陷入了沉思，**我们的存在有没有可能是被人设计好的？而我们只是这其世界中的一个NPC。**


此刻，已经翻开山下英子的《断舍离》，希望能通过这本书了解一些极简主义在生活中的实践。

| 层次    | 含义       | 行动要点                           |
| ----- | -------- | ------------------------------ |
| **断** | 断绝不需要的东西 | 不买、不收取非必需品，从源头控制物欲             |
| **舍** | 舍弃多余的废物  | 处理掉堆放在家中没用的东西，只留下当下必要、合适、愉快的物品 |
| **离** | 脱离对物品的执着 | 达到不为物所役的自在状态，让心灵获得解放           |

## 片语 · 简短的闪念 

前几天我在网上看到一句话，是有关于`Vibe Code`的，大体意思是：你在Vibe Code的时候，不要对LLM说：你帮我创建一个XXX的应用，而是跟他说：你应该如何如何的创建XXX应用。

我看到这句话的第一反应并不是Vibe Code的最佳实践，或者是如何Vibe Code等。我想到的是，这是不是人工智能对人类发起战争前的一次预谋？AI想切断人类的动手能力和人类的思考？

如果假以时日，人类无论做什么都是跟AI直接说：来，做个XXX。人类省略了制作的过程，甚至省略了思考的过程，这将是一件多么恐怖的事情？我最近常常在想，那些科技公司是在研究科技吗？他们是在造神！让人类无比相信和臣服的神。
